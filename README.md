# Feedback Widget

A React component for collecting user feedback with screenshot and screen recording capabilities, plus an annotation tool.

## Features

- **Screenshot capture** - Capture the current screen/tab
- **Screen recording** - Record screen activity
- **Annotation tool** - Draw, add text, and annotate screenshots

## Usage

The widget provides a simple interface for users to report issues or provide feedback. Users can capture screenshots, record their screen, and annotate images with freehand drawing or text annotations.
